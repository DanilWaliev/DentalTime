# Клиент-серверное веб-приложение для автоматизации записи пациентов на оказание стоматологических услуг

*Проект разрабатывается в рамках учебного задания по дисциплине «Интернет-технологии» СПбГТИ (ТУ)*

## Ключевые возможности

### Пациент
1. **Поэтапный выбор** услуги, врача, даты и времени для формирования записи.
2. **Управление созданными записями:** изменение и отмена.
3. **Просмотр** каталога услуг.
4. **Просмотр** персонала клиники.

### Менеджер
1. **Управление базой персонала и каталогом услуг:** создание, просмотр, изменение и удаление.
2. **Управление созданными записями:** подтверждение, изменение и отмена.
3. **Просмотр записей** с помощью интерактивного расписания с возможностью изменения или отмены.

## Использованные технологии

### HTML / CSS
* **Макет:** CSS Grid и Flexbox.
* **Стилизация:** Box-shadow, CSS-переменные, Transition.
* **Адаптивность:** Responsive design.
